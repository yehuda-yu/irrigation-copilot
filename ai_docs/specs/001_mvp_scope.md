# MVP Scope

## Overview
Minimal viable product for irrigation recommendations.

## Features
- Single location (lat/lon) input
- MoAG forecast data source
- Basic Kc-based computation
- Text output (farm mode: mm/day, liters/dunam; home mode: ml/day)

## Out of Scope (Future)
- Multi-location batch processing
- IMS official station API
- LangGraph workflow with human-in-the-loop
- Advanced scheduling algorithms
